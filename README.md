# 码搭 CodeForge — AI 前端页面智能搭建助手

> 黑客松参赛项目 | 输入需求 → AI生成 → 在线预览

## 快速启动

```bash
# 1. 安装依赖
npm install

# 2. 启动开发服务器
npm run dev

# 3. 打开浏览器访问
# http://localhost:3000
```

## 核心流程

```
用户输入业务需求（自然语言）
→ AI分析需求（识别页面类型、字段、筛选、操作）
→ 生成页面结构（列表页/表单页/详情页）
→ 生成 Vue3 + TypeScript 代码
→ 页面在线预览（可交互）
```

## 功能特性

- **需求转页面**：输入自然语言描述，自动生成完整页面
- **代码生成**：生成 Vue3 + TypeScript SFC 代码，使用企业组件库
- **在线预览**：生成的页面可实时交互预览
- **代码展示**：带语法高亮的代码展示，支持一键复制
- **Mock数据**：自动生成演示数据
- **模拟组件库**：内置 CfTable、CfForm、CfButton 等企业级组件

## 技术栈

- Vue 3.4 + TypeScript
- Vite 5
- Pinia（状态管理）
- Highlight.js（代码高亮）
- 运行时模板编译（vue/esm-bundler）

## 演示示例

在输入框中输入以下需求，点击生成：

1. **列表页**：`创建一个设备管理列表页，展示设备编号、设备名称、设备类型、状态、所属车间、最后维护时间，支持按名称和状态筛选，支持新增和编辑`

2. **报警管理**：`创建一个设备报警管理页面，展示报警编号、设备名称、报警等级、报警时间、报警内容、状态，支持按设备、等级筛选，并支持查看详情和关闭报警`

3. **表单页**：`创建一个工单录入表单，包含工单标题、工单类型、优先级、描述、负责人字段`

4. **详情页**：`创建一个设备详情页面，展示设备编号、设备名称、设备类型、状态、所属车间、负责人、创建时间、最后维护时间`

## 支持的页面类型

| 类型 | 关键词 | 生成内容 |
|------|--------|---------|
| 列表页 | 列表、展示、管理 | 表格 + 筛选 + 操作按钮 + 分页 |
| 表单页 | 表单、录入、填写 | 表单项 + 提交/重置 |
| 详情页 | 详情、详细 | 字段展示 + 返回 |
